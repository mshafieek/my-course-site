Introduction to Machine Learning and Data Analysis

Welcome to the website for the one-week Summer School course Introduction to Machine Learning and Data Analysis in R. This course offers a practical introduction to key methods in machine learning and data analysis using real datasets and hands-on examples. Participants are introduced to topics such as supervised and unsupervised learning, model validation, handling missing data, and presenting results in a transparent and reproducible way. The emphasis is on application rather than theory, with a focus on clear, interpretable methods that can be used in research practice. Here you can find the daily course materials for Monday through Friday.

R packages

In this course, we use a wide range of R packages. It is highly recommended that you have administrator rights on your laptop, because this will allow you to install these packages when needed.

Otherwise, you should install these packages before the start of the course.

You can install them with the following code:

pkgs <- c("tidyverse","knitr","rmarkdown","patchwork","palmerpenguins","caret","ISLR",
          "glmnet","randomForest","xgboost","e1071","kernlab","pROC","psych",
          "tidytext","factoextra","neuralnet")

to_install <- setdiff(pkgs, rownames(installed.packages()))
if (length(to_install)) install.packages(to_install, repos = "https://cloud.r-project.org")

If you do not have administrator rights and were not able to install the necessary software or packages, you can:

  • work with RStudio via MyWorkPlace using your Utrecht University account
  • install Citrix Workspace to access MyWorkPlace
