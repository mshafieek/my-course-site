Introduction to Machine Learning and Data Analysis

Welcome to the website for this one-week Summer School course.

In this course, participants work with real datasets and follow a complete research workflow: from data preprocessing to supervised and unsupervised learning, model validation, handling missing data, and presenting results in a transparent and reproducible way.

The course is designed to make machine learning a practical tool for research, using clear, applied examples and a strong focus on interpretable methods.

Software

You need to have installed the following on your laptop, in this order:

  1. R
  2. RStudio

R packages

In this course, we use a wide range of R packages. It is highly recommended that you have administrator rights on your laptop, because this will allow you to install these packages when needed.

Otherwise, you should install these packages before the start of the course.

You can install them with the following code:

install.packages(c("tidyverse", "knitr", "rmarkdown", "patchwork", "palmerpenguins", "caret", "ISLR", "splines", "glmnet", 
      "randomForest", "xgboost", "e1071", "kernlab", "pROC", "psych", "tidytext", "factoextra", "neuralnet"
))

If you do not have administrator rights and were not able to install the necessary software or packages, you can:

  • work with RStudio via MyWorkPlace using your Utrecht University account
  • install Citrix Workspace to access MyWorkPlace
